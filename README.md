# lvlup-archive

Archiwum oficjalnego forum lvlup.pro

Instalacja i uruchomienie:

```bash
git clone https://github.com/OreQr/lvlup-archive

cd ./lvlup-archive

wget https://github.com/OreQr/lvlup-archive/releases/download/forum/forum.zip // Pobranie forum
unzip forum.zip

npm install

npm run build

npm run preview
```
