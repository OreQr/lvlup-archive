# lvlup-archive

Archiwum oficjalnego forum lvlup.pro

Instalacja i uruchomienie front-endu:

```bash
git clone https://github.com/OreQr/lvlup-archive

cd ./lvlup-archive/apps/www

wget https://lvlup.oreq.xyz/forum.zip // Pobranie forum
unzip forum.zip

npm install

npm run build

npm start
```
